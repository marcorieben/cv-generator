---

## Workflow

1. **Start a Feature:** Copy TEMPLATE.md, customize for your feature, save as `FEATURE_NAME.md`
2. **Track Progress:** Update checkbox as you complete tasks; commit changes with implementation
3. **Complete Feature:** Mark all boxes; merge branch and archive TODO file (or move to COMPLETED folder if needed)
4. **Roadmap Updates:** Reflect completed milestones in ROADMAP.md
